# Habit Tracker Dashboard

Professional grey/beige themed habit tracker with analytics.

## Features
- ✅ 31-day habit tracking grid
- 📊 Daily & Weekly progress charts
- 📈 Overall statistics with donut chart
- 📋 Detailed analysis table
- 🏆 Top 10 habits ranking
- 😊 Mood & Motivation tracking
- 📉 Trend line chart
- 💾 LocalStorage save

## How to Use
1. Open `index.html` in browser
2. Click checkboxes to mark habits complete
3. Add new habits from sidebar
4. Data auto-saves in browser

## Color Scheme
- Background: #e8e4dc (Warm grey/beige)
- Cards: #c8c4b8 (Darker beige)
- Text: #3a3832 (Dark grey)
- Accents: #5a5852 (Charcoal)
